# UCMJ – HTML Edition (Chapters 1–12)

This folder is a static site with 12 linked chapter pages and a homepage (index.html).

## Local Viewing
Double‑click `index.html` (or any `chapter-XX.html`) to open in your browser.

## Deploy (Netlify)
1. Create a new site from your dashboard.
2. Drag & drop this entire folder.
3. Done — no build settings needed.

## Deploy (GitHub Pages)
1. Create a repo and upload the folder contents.
2. Enable GitHub Pages (Settings → Pages → Source: `main` → `/root`).
3. Your site will be live at the provided URL.

## Structure
- `index.html` — homepage + table of contents
- `chapter-01.html … chapter-12.html` — individual chapters with prev/next nav
- `assets/style.css` — shared styles
